Magic Card API

Usage

Use the endpoint https://magic-tcg-api.freecodecamp.rocks/api/cards to see a list of all valid magic card names and ID numbers.

Use the endpoint https://magic-tcg-api.freecodecamp.rocks/api/cards/{name-or-id} to get data for a card, where {name-or-id} is the card's name or ID number.
